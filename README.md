
# 📊 Amazon Sales Data Analysis (Python Project)

## 🚀 Overview

This project generates and analyzes Amazon sales data using Python and Pandas. It creates a structured dataset and exports it into an Excel file for further analysis and visualization.

## 📌 Features

* 📦 Generates sample e-commerce sales data
* 🧮 Calculates total sales automatically
* 📊 Exports data to Excel format
* 🗂 Organized dataset with categories, regions, and payment modes

## 🛠 Technologies Used

* Python
* Pandas
* Excel (for output)

## 📁 Project Structure


amazon-sales-data-analysis/
│── amazonsales.py
│── amazon_sales_dataset.xlsx
│── requirements.txt

```

## ▶️ How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the script:

   ```
   python amazonsales.py
   ```

## 📈 Output

* Generates `amazon_sales_dataset.xlsx`
* Includes total sales column
